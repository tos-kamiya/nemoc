# nomoc, a command-line rendering tool for NEMO template

A CLI tool to render nemo template (experimental).

# dependencies

- nemo https://github.com/9cloud/Nemo

# usage

Give nemoc.py a nemo template.

```bash
$ echo %p | python nemoc.py
<p />
```
